4 Channels Temperatures Monitor


Introduction
Ce projet est un moniteur de temperatures hardware. Il est concu a la base pour les PC, mais rien ne l'empeche de fonctionner dans un autre environnement qui necessiterait le meme genre de caracteristiques. Les caracteristiques en sont les suivantes :
  • 4 sondes de temperature possible
  • Plage de mesure : 10-80°C
  • Reolution : 0.1°C
  • Precision : 0.5°C
  • Affichage alphanumerique 2 lignes de 16 caracteres chacune
  • Nom de chaque sonde librement definissable (9 caracteres)
  • Alarme sonore reglable pour chaque sonde
  • Possibilite d'extinction automatique du PC en cas de surchauffe, reglable aussi pour chaque sonde
  • Affichage sur ecran alphanumerique 2*16
  • 4 bouttons permettent de naviguer aisement dans les menus
  • Totalement independant du software du pc (autonome)
Description
Le coeur du projet est un microcontroller de chez Microchip, un PIC 16F876A. Il gere tout le montage : affichage, touches, conversion analog/digitale des temperatures, activation des alarmes et extinction du PC si necessaire.
Tous les reglages se font par menu interactifs, affiches a l'ecran. Il est possible de definir le nom de chaque sonde (CPU, Mainboard par ex), de regler la valeur qui declenchera l'alarme, celle qui entrainera l'extinction du PC (ou de tout desactiver) independament pour chaque sonde, l'activation ou nom du backlight du LCD (pour les versions avec backlight).
Pourquoi un montage autonome ? Le but etant d'obtenir la plus grande fiabilite et compatibilite possible, ainsi que de preserver les ressources de la machine. Fiabilite car le montage est autonome, il n'a besoin d'aucune ressource du PC pour fonctionner. Meme si ce dernier plante, l'extinction de l'ordinateur se fait en 5s maximum (cette option necessite une alimentation de type ATX). Comme il est totalement independant du materiel qui compose votre configuration, il est compatible avec toutes les machines existante. Encore un avantage d'un systeme autonome, c'est qu'il ne consomme aucune ressource de l'ordinateur (CPU, RAM, ...), tout est gere sur la carte et ne diminue en aucun cas les performances de la machine.
Systeme autonome siginie aussi -comme mentionne plus haut- que ce montage peut se voir utilise dans d'autres cas que sur un ordinateur, pour peu qu'on lui fournisse l'alimentation necessaire bien evidement.
